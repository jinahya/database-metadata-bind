# JAVA

Any given Java file should have the following elements ordered as specified.

Single line hyphen comment has margin of 120,
and should exist even no elements are specified

```text

serialVersionUID if exists

// ---------------------------------- COMPARATORS

// ---------------------------------- {COLUMN_NAME1}
static fields related to {COLUMN_NAME1}

// ---------------------------------- {COLUMN_NAME2}
static fields related to {COLUMN_NAME2}

// ------------------------- STATIC_FACTORY_METHODS

// ------------------------- CONSTRUCTORS

// ---------------------------- java.lang.Object
toString
equals
hashCode

// ------------------------------- Jakarta-Validation

// --------------------------------- {field_name1}
getter
setter

// --------------------------------- {field_name2}
getter
setter

// --------------------------------------------------
instance fields
```
