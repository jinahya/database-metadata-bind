# AI_

## AI_CONTEXT.md (설계 및 규칙)

### 역할

프로젝트의 "헌법"과 "설계도".

### 포함 내용

기술 스택(Next.js, Tailwind 등), 코딩 스타일 규칙, 디렉토리 구조, 주요 API 엔드포인트 정보.

### 사용법

AI가 엉뚱한 라이브러리를 쓰거나 내 스타일과 다른 코드를 짜지 않도록 가이드를 잡는 용도입니다.

## AI_PROGRESS.md (진행 상황)

### 역할

프로젝트의 "작업 일지" 및 "체크리스트".

### 포함 내용

현재까지 완료된 기능, 현재 작업 중인 이슈, 다음에 해야 할 일(To-do).

### 사용법

대화가 끊겼을 때 "어디까지 했더라?"를 방지합니다. AI에게 **"방금 한 일 progress에 반영해줘"**라고 시키면 좋습니다.

## AI_SUMMARY.md (압축된 기억)

### 역할

대화창을 새로 갈아탈 때 쓰는 "환승용 티켓".

### 포함 내용

지금까지의 논의 결정 사항 중 핵심 로직 요약, 복잡한 트러블슈팅 결과물.

### 사용법

대화가 너무 길어져서 토큰 제한이 오면, AI에게 **"지금까지 내용을 AI_SUMMARY.md에 요약해줘"**라고 한 뒤, 이 파일 내용만 복사해서 새 대화창에 붙여넣고 시작합니다.
